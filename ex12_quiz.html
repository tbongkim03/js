태조 이성계가 태어난 년도는?

(응용 1)


위에서 1.2를 여러번 곱했는데 1.2를 10번 곱하려면 코드를 어떻게 짜야될까요?
1.2 * 1.2 * 1.2 ... 계속하면 되긴 하는데
자바스크립트엔 ** 이런 거듭제곱 연산자 기능도 있습니다.


        var a = 1.2;
        var result = a * a * a;
        var result2 = a ** 3;

(응용 2)


커피리필 문제에서
커피리필을 무한으로 해준다면 처음 담아주는 커피가 360ml일떄 총 몇 ml의 커피를 마실 수 있을까요?
무한등비수열의 합 그런건데 공식이 가물가물해서 여기까지 하겠습니다.


        var 처음마신커피양 = 360;
        const 리필비율 = 2/3
        var 무한등비수열합공식 = 처음마신커피양 / (1 - 리필비율) // S = 3 * 처음마신커피양 ... 1080 ml>
